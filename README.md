# TreeCutter

[![Available for Paper](https://raw.githubusercontent.com/Fluxoah/Banners/main/available_for_paper.png)](https://modrinth.com/plugin/simpletreecutter)
[![Available on Modrinth](https://raw.githubusercontent.com/Fluxoah/Banners/main/available_on_modrinth.png)](https://modrinth.com/plugin/simpletreecutter)

TreeCutter is a lightweight and configurable Minecraft plugin that lets players cut down entire natural trees instantly by sneaking and breaking the bottom log.

**Made by Mistix**

</div>

---

## Features

- Instant tree cutting when sneaking and breaking the bottom log.
- Cooldown support to prevent abuse.
- Configurable behavior via `config.yml`.

---

## Installation

1. Build the plugin:
   - `gradlew.bat clean build` (Windows)
2. Place the generated JAR in your server `plugins` folder:
   - `build/libs/mistix-treecutter-paper-1.0.0.jar`
3. Start the server once to generate default config.
4. Edit `config.yml` as needed and restart.
